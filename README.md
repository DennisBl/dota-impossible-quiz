# The Impossible Dota Quiz

A single-page browser game inspired by *The Impossible Quiz* — 20 Dota 2 riddles
built on misdirection, wordplay, and lateral thinking. Three lives, three skips.

## Play

Open `index.html` in any modern browser. No build step, no dependencies.

## Features

- 20 trick questions with twists (hidden answers, click hotspots, timed clicks,
  sequence puzzles, moving tiles)
- Lives and skip system
- Atmospheric ember-particle UI

## Tech

Plain HTML/CSS/JS in a single file.
